LightTank

LightTank is a collaborative project with the arc/sec lab in School of Architecture and Planning, University of Auckland. In this project, we implemented a multi-user, large scale touch-tracking system using multiple off-the-shelf Lidar scanners. The final installation is a composite wall structure with interactive stereoscopic 3D visualizations that is responsive to users touching the walls. The installation is part of the Ars Electronica 2018, one of the premier avenues that honor worldwide sensations.
