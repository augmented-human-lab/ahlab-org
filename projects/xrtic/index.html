XRtic·2022

XRtic is a prototyping toolkit enabling real-world cloth deformations to be used in novel ways in eXtended Reality (XR) applications. XRtic was developed based on the insights gathered from semi-structured interviews with XR developers. It is a fully functional implementation which consists of custom-made actuators that can be attached to regular clothing, a controller bus system, and a controller interface. Using the XRtic toolkit, users can design and integrate different cloth deformation types synchronised with virtual content in a plug-and-play manner. XR developers can use XRtic to trigger cloth deformation based on particular actions or states of an XR application. We believe XRtic will provide a substantial commencement for the XR community to use cloth deformations in novel ways with XR applications.
